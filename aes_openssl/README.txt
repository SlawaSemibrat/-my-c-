Программа считывает текст в файле input.txt и записывает результат шифрования в output.txt. 
Затем считывает из output.txt, и расшифровывает. Расшифрованный текст выводится в консоль.(Перед запуском следует поменять путь до файлов input.txt и output.txt в коде)
aes_openssl.cpp - код
input.txt - файл для ввода ткста
output.txt - файл для зашифрованного текста
aes-output - результат выполнения программы. Можно открыть открыть командой ./3laba-output
